# 2.Git
Git倉庫管理功能是GitHub的核心，所以在用GitHub前要先了解Git
* 版本管理系統：紀錄程式碼的變更，提供回朔功能
  * 中心式：程式碼集中在單一伺服器中
    * 優點：便於管理
    * 缺點：伺服器掛掉就無法獲取最新程式碼，甚至遺失程式碼
  * 分散式：每個用戶可以有自己的本地倉庫
    * 優點：不必連接伺服器就可開發
    * 缺點：學習門檻
